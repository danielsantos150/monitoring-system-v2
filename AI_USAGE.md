# AI Usage Note

AI tools were used as supporting resources during the completion of this assessment, primarily for brainstorming, reviewing solution approaches, improving documentation structure, and refining written explanations.

### Tools Used

* ChatGPT
* Claude Code
* Gemini

### Types of Prompts Used

Examples of prompts included:

* Reviewing and validating architecture decisions.
* Generating alternative approaches for server monitoring and deployment.
* Improving the clarity and structure of written responses.
* Discussing database design and monitoring strategies.
* Reviewing deployment, rollback, and feature flag approaches.
* Assisting with code scaffolding and documentation formatting.

### What Was Accepted vs. Discarded

Accepted:

* General architectural recommendations.
* Documentation structure suggestions.
* Alternative implementation approaches.
* Code examples used as references.

Discarded:

* Generic or overly complex solutions that did not fit the scope of the assessment.
* Recommendations that introduced unnecessary infrastructure or dependencies.
* Responses that did not align with the proposed system design.

### Validation Process

All generated content was reviewed and adapted before inclusion.

Validation included:

* Verifying technical feasibility based on professional experience.
* Ensuring consistency with the proposed database model and application architecture.
* Testing and reviewing the implemented code.
* Adjusting explanations to match the final implementation.

### External Sources and Documentation

In addition to AI tools, standard technical documentation and general industry practices were referenced, including:

* PHP Documentation
* Docker Documentation
* MySQL Documentation
* Python Packaging Documentation
* General software deployment and monitoring best practices

# Monitoring System

## Technologies

- PHP 8.3
- MySQL 8
- Docker
- HTML
- CSS
- JavaScript

---

## Architecture

Browser
    ↓
JavaScript (Fetch API)
    ↓
PHP API Layer
    ↓
Repository Layer
    ↓
MySQL Database

## Features

- Server CRUD
- Service CRUD
- Alert Rules CRUD
- Alert Management
- Dashboard
- Infrastructure Metrics
- Charts with Chart.js
